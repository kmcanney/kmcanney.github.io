Cutting sound data

Choose Edit > Cut to remove the selected sound data and copy it to the clipboard. Cutting sound data replaces the previous contents of the clipboard.

For information about cutting events, please see "Cutting, copying, pasting, mixing and deleting events."

  1. Choose Edit > Mode > Time (or click the Time Mode button in an editor's playbar).

  2. Select the data you want to cut. For more information, see "Selecting data using the mouse."

  3. Choose Edit > Cut.

    If you cut data from individual channels of multichannel files, the waveform will contain silence at the end of the cut channel. The channels in a multichannel file must always be equal in length.

Exporting regions

Choose File > Export Regions to create new files from regions in your file.

  1. Choose File > Export Regions. The Export Regions dialog is displayed.

  2. Type a name in the Prefix box if you want add a prefix to extracted regions. For example, type Test to extract Test_1.wav, Test_2.wav, Test_3.wav, and so on.

  3. Choose a location from the Where drop-down list to choose where you want to export your files.

  4. Choose a file format from the Format drop-down list to choose the kind of files you want to export.

  5. Choose a setting from the Preset drop-down list to choose the attributes that will be used to export your files.

  6. A list of regions is displayed at the bottom of the dialog. Select the Export check box for each region you want to export, or click Select All to export all regions.

  7. Click the Export button.

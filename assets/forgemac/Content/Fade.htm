Fade in/out

Choose Process > Fade In or Fade Out, and then choose a fade type from the submenu to vary the volume of a sound file over time.

  1. Select the data you want to fade. If no data is selected, the fade will be applied to the entire file.

    For more information, see "Selecting data using the mouse."

  2. Choose Process > Fade In or Fade Out, and then choose a fade type from the submenu.

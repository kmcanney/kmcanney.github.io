Creating a new sound file

Each sound file is opened in a separate tab in the editor. You can have multiple files open at once; click anywhere in a tab to make it active.

If you want to create a new sound file using a section of another file, you can use the Paste to New command. For more information, see "Paste to New."

  1. Choose File > New.

  2. Use the New Sound File dialog to specify the parameters for the new file:

    1. Choose a sample rate from the Sample rate drop-down list.

    2. Choose a setting from the Bit depth drop-down list to specify the number of bits that should be used to store each sample.

    3. Choose a setting from the Channels drop-down list to specify the number of channels that will be used in the window.

      If you want to use the same settings when you create new files, you can select the Use these settings for all new files check box or use the Use default settings for new files check box in the Preferences dialog.

  3. A new, untitled sound file is created.

Keyboard shortcuts

The following shortcut keys that can help streamline your work with Sound Forge Pro software. The available shortcut keys are arranged in tables according to function.

If you want to customize your keyboard shortcuts, you can use the Keyboard Shortcuts page in the Preferences dialog. For more information, see "Preferences."

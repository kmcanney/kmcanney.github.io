Sound Forge Pro 2.0 provides a contemporary application environment that’s perfect for recording, editing, processing, and rendering broadcast-quality audio master files. A fresh statement on audio editing software, Sound Forge Pro Mac includes a comprehensive suite of mastering, restoration, and repair tools for immediate results.

What's new in version 2.0

  • Added loudness meters and loudness logging.

  • Added support for ripple editing in event-editing mode. Choose Options > Event > Auto Ripple to toggle automatic ripple editing for downstream events.

  • Added support for moving markers, regions, and envelope points with event and audio selections. Choose Options > Lock to Selection > Markers/Regions and Options > Lock to Selection > Envelope Points to toggle marker/region and envelope-point locking.

  • Added support for splitting events at the cursor or region boundaries.

  • Added support for snapping selections to zero crossings.

  • Added support for automatically trimming audio.

  • Added a customizable editing toolbar.

  • Added Create new region and Create new window recording modes.

  • Added support for drag-and-drop event pasting and mixing.

  • Added support for editing files in SpectraLayers Pro 2.0.

  • Added support for reading and saving FLAC audio.

  • Added support for pasting and mixing events by holding Command while moving events by dragging them.

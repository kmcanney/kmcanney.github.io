Archiving projects

Choose File > Export >Vegas Project Archive to export your project and its associated media.

  1. Choose File > Export >Vegas Project Archive. The Vegas Project Archive dialog is displayed.

  2. Use the Vegas Project Archive dialog to choose options for saving your archive:

    1. In the Project file box, choose the folder and file name you want to use to save your archive.

    2. Select the Include nested projects check box if you want to include nested projects in the archive.

    3. Select the Include media if you want to copy media files with the archive.

    4. Select the Include video proxy/audio peak files check box if you want to include audio peaks and audio/video proxy files with the archive. For more information, see "Rebuild audio peaks" or "Creating proxy files for high-definition editing."

    5. Select the Exclude unused media if you want to skip archiving unused media files. If you clear the check box, all project media will be archived.

  3. Click the Finish button to create your project archive.

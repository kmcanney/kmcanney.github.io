Updating Project Video Settings to Match a Video File

Vegas Pro makes it quick and simple to update your video Project Properties based on an existing media file.

  1. From the File menu, choose Properties to display the dialog.

  2. On the Video tab, click the Match Media Settings button . The Open dialog is displayed.

  3. Browse to the file you want to match and select it. The properties for the selected file are displayed at the bottom of the dialog.

  4. Click Open. The settings in the Project Properties dialog are updated to match the file you selected.

    For more information, see "Setting project properties."

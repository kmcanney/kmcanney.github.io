Saving a Still Image from the Timeline

  1. Position the cursor in the timeline to choose the frame you want to save. The frame is displayed in the Video Preview window. For more information, see "Using the Video Preview window."

  2. Click the Video Preview Quality button and choose a setting from the menu to change the size of the frame in the Video Preview window.

  3. Click the Save Snapshot to File button to save the current frame as an image file.

  4. The Save Snapshot to File dialog is displayed to allow you to choose the format and location where you want to save the file. The file is automatically added to the Project Media list. For more information, see "Using the Project Media window."

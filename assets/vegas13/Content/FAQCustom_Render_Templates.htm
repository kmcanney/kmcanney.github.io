Where Are My Custom Rendering Templates?

Rendering templates are stored in C:\Users\<username>\AppData\Roaming\Sony\Render Templates\<plug-in name>.

The AppData folder is not visible unless the Show hidden files and folders radio button is selected on the View tab of the Windows Folder Options control panel.

You can find a plug-in's name by clicking the About button in the Render As dialog.

To make a template available on another computer or user account, copy the .sft2 file to the same location in another account.

For more information about creating custom templates and transferring them across computers or user accounts, see "Custom rendering templates."

Merging Vegas Pro Projects

Although you can now nest Vegas Pro projects within other projects, there may be times that you want to merge two existing projects into a single project file. For more information about nesting projects, see "Nesting projects."

  1. Open two instances of Vegas Pro and open the projects you want to merge.

  2. Switch to the project you want to merge and press Ctrl+A to select all events.

  3. Click the Copy button .

  4. Switch to the other instance of Vegas Pro and position the cursor at the point where you want to merge the project you copied in step 3.

  5. Click the Paste button .

  6. Click the Save button to save your merged project, or click Save As if you want to save the merged project with a new name.

You can also drag media bins from the Project Media window across two instances of Vegas Pro. Drag a bin from one Project Media window and drop it on a bin in the other instance. For more information, see "Using the Project Media window."

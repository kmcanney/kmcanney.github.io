Applying an Effect to a Portion of a Video Event

The short answer is that effects cannot be applied to only a portion of an event, but that's not exactly what you were hoping to hear, so there are a couple of methods you can use to achieve the result you're looking for.

Splitting the event

  1. Select the event.

  2. Create a time selection that includes the portion of the event where you want to apply the effect. For more information, see "Creating selections and positioning the cursor."

  3. Press S to split the event at each end of the selection. For more information, see "Splitting events."

  4. Add the effect as an event effect to the event you created by splitting. For more information, see "Adding video effects."

    Add a crossfade between the affected and unaffected events to fade the effect in and out. For more information, see "Automatic crossfades."

Using effect automation

  1. Add a plug-in to an event as an event effect. For more information, see "Adding video effects."

  2. Select the Sync Cursor button in the keyframe controller at the bottom of the Video Event FX dialog.

  3. In the timeline, click to position the cursor at the point where you want the effect to begin.

  4. In the Video Event FX dialog, choose a preset or adjust the plug-in controls as desired.

    A new keyframe will be created at the cursor position in the keyframe controller.

  5. In the timeline, click to position the cursor at the point where you want the effect to end.

  6. In the Video Event FX dialog, choose the Reset to None preset.

    A new keyframe will be created at the cursor position in the keyframe controller.

    When you play the event, the effect will be applied in the portion of the event between the two keyframes you created.

    For more information about using keyframes, see "Keyframe animation."

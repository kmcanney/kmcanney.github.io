Removing an assignable effects chain

When you remove an assignable effects chain, it is deleted from the project. You must recreate the chain if you want to use it again. If you want to keep the chain, but do not want to use it in your current project, you may save the chain as a preset. For more information, see "Creating and using effects packages."

To bypass an assignable effects chain without removing it from the project, select the Mute button in the Assignable FX Control.

  1. In the Mixing Console window, select the assignable effects chain you want to remove (to select multiple chains, hold down the Ctrl key while selecting).

  2. Right-click a selected chain and choose Delete from the shortcut menu (or press Delete). The effects chains are removed from your project.

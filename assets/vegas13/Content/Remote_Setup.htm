Setting up Vegas Pro Connect

Vegas Pro Connect saves connection profiles that allow you to use the app with a number of computers running Vegas Pro.

  1. Enable the Vegas Pro Connect host on each computer you want to connect to.

    1. In Vegas Pro, verify Options > External Control is enabled.

    2. In Vegas Pro, choose Options > Preferences and select the External Control & Automation tab.

    3. Choose Vegas Pro Connect from the Available devices drop-down list and click the Add button.

    4. Double-click the Vegas Pro Connect entry in the Active control devices list.

    5. The Configure Vegas Pro Connect dialog displays a list of possible addresses for your computer and the port that will be used to communicate with your computer.

      Record the host addresses and port number. You'll need them when you set up Vegas Pro Connect.

      6. Your mobile device and the computer running Vegas Pro must be connected to the same network.

    6. Click OK to close the Configure Vegas Pro Connect and Preferences dialogs.

  2. Set up a connection in Vegas Pro Connect:

    1. Start Vegas Pro Connect on your mobile device.

    2. Tap Set up Vegas Pro connection to create a new connection, or tap to edit an existing connection.

    3. Use the Set Up Vegas Pro Connection settings to name your connection and specify the address and port of the computer you noted in step 1e above.

    4. Tap Try It! to test the connection.

      If the connection is successful, Success is displayed.

      If the connection fails, you may need to adjust your network or connection settings. For more information, see "Troubleshooting Vegas Pro Connect Connections."

    5. Tap Save.

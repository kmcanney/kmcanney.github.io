Rendering Projects for Use in DVD Architect Pro

If you have DVD Architect Pro software, you can avoid unnecessary transcoding or recompression by using Vegas Pro to render your media files in the appropriate formats prior adding them to your DVD Architect Pro project.

The Sony Creative Software Inc. AC-3 and MainConcept MPEG-2 encoders provide templates that will produce files that do not require recompression (unless the file size is too large to fit on a DVD).

Saving a project

From the File menu, choose Save to save changes to the current Vegas Pro project (.veg). When a Vegas Pro project is saved, all of the layout of tracks, envelope adjustments, effects parameters, etc. are saved in this project file.

The first time you save a project, the Save As dialog is displayed.

If you want to keep the previous version of your project before you made changes, use the Save As command to save the new version of the project with a new name. For more information, see "Saving and renaming a project (Save As)."

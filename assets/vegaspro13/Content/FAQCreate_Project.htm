Creating Projects in Vegas Pro

Vegas Pro is an incredibly powerful audio and video editor, but don't let that intimidate you: it's very easy to get started creating Vegas Pro projects.

  1. Create a new project file. For more information, see "Creating a new project."

    A project file is like your canvas and palette: it's not a multimedia file, but it contains information about all the media in your project, how the media is arranged, effects, and so on. A project file lets you edit without worrying about changing your source media files.

  2. Add media to the project. Your media can come from any number of sources:

    • You can add media files from your computer. For more information, see "Adding media files to your project."

    • You can record audio for dialog, voiceovers, music tracks, or sound effects. For more information, see "Recording audio."

    • You can capture video from a video camera. For more information, see "Capturing video."

    • You can import video from a DVD camcorder. For more information, see "Importing video from a DVD camcorder."

    • You can download media from the Web. For more information, see "Getting media from the Web."

  3. Create events on the timeline to indicate where you want to play each media file. For more information, see "Adding media files to your project."

    You can drag files from the Explorer window, Project Media window, or Windows Explorer to the Vegas Pro timeline to create events.

  4. Edit the events on the timeline as needed. For more information, see "Editing events on the timeline."

    You can move events, adjust their length, crop video, or edit properties to fine-tune the look and sound of your project.

  5. Save your project. For more information, see "Saving a project."

  6. Deliver your project in its final format:

    • You can render a file if you want to create a video or audio file. For more information, see "Rendering projects (Render As)."

    • You can burn CDs. For more information, see "Burning discs."

    • You can save a movie to your DV or HDV video camera. For more information, see "Printing video to tape."

    • You can burn a movie to DVD or Blu-ray Disc. For more information, see "Burning discs."

  7. You can burn a movie to DVD or Blu-ray Disc. For more information, see "Burning discs."

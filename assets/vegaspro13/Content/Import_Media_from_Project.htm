Importing media from a project file

From the File menu, choose Import, and then choose Media from Project from the submenu to add media from another Vegas Pro project file to your project.

Using this feature, you can create small, template projects to streamline frequent editing tasks.

Generated media and offline media files are not imported.

  1. From the File menu, choose Import, and then choose Media from Project from the submenu. The Import Media dialog box is displayed.

  2. Choose the folder where the file you want to open is stored:

    • Choose a drive and folder from the Look in drop-down list.

    • Choose a folder from the Recent drop-down list to quickly select a folder from which you have previously opened files.

  3. Select a file in the browse window or type a name in the File name box.

  4. If you want to import the project's media bins, select the Import bins check box. For more information, see "Sorting your media with bins."

    Select the Merge with existing bins check box if you want to merge the imported project's bins with the current project. When the check box is cleared, a bin is created using the imported project's name, and the project's bins are imported inside that bin.

  5. Click Open.

  6. The project's media is added to the Project Media window. For more information, see "Using the Project Media window."

Trimming events

Click the Trim button on the editing toolbar to remove all data from the events except the selected data.

This command has no effect if there is no selected data.

  1. Drag the cursor to make a timeline selection (or hold down the Shift key and use the arrow keys to make the selection).

  2. Click the Trim button on the editing toolbar. All data outside of the selection is deleted. Events that are not selected will not be trimmed.

    Tips:

    • Click the Trim Start button (or press Alt+[) to trim the start of the event to the cursor position.

    • Click the Trim End button (or press Alt+]) to trim the end of the event to the cursor position.
